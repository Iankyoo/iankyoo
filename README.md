## ◆ Projetos em destaque 🚀

### Bank Analyzer

API REST inteligente para processamento e análise de extratos bancários utilizando IA.

#### Features
- Upload e processamento de PDFs bancários
- Extração automatizada de transações
- Categorização financeira com LLMs
- Memória semântica com embeddings vetoriais
- Autenticação JWT
- Arquitetura assíncrona
- Deploy containerizado

#### Stack do projeto
`Python` · `FastAPI` · `PostgreSQL` · `SQLAlchemy Async` · `LangChain` · `Gemini` · `ChromaDB` · `Docker` · `AWS S3`
